## Application Usage
- XAMPP is REQUIRED! 
- git clone inside xampp\htdocs
- run apache and mySQL server
- run mySQL script
- run "http://localhost/UAS-Basdat/" into your browser!

_Features:_  
- 
- User-to-Merchant payment simulator and tracking
- Merchant simulator for withdrawing and tracking
- Admin system simulator for blocking and logging
